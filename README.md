## 🚀 Getting started

Clone the project and access the folder.

```bash
$ git clone https://github.com/maarvins/maarvins.dev.git && cd personal-landingPage
```

Follow the steps below
```bash
# Install the dependencies
$ npm install


# Start the project
$ npm start
```
